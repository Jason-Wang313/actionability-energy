\section{Conclusion}

Actionability energy formalizes a simple requirement for embodied world models: imagined representation changes should lie near the local action-reachable tangent set of the body that must execute them. In controlled CPU benchmarks, this residual predicts execution failure, supports inference-time repair, and changes under embodiment swaps. The result is not a replacement for video planners, inverse dynamics, or real robot evaluation. It is a diagnostic and repair factor that can make generated futures more honest about what an embodiment can locally do.
